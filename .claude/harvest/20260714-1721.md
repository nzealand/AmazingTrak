# Harvest — 2026-07-14 (vantage spots feature session)

## INVARIANTS

- **Admin moderation features get a dedicated, never-reused permission level number.** `requirePermission(N, ...)` grants access when `PermissionLevel >= N` (or `PermissionLevel == 0` for the central admin), so levels form a hierarchy, but this codebase's convention is one level per feature domain rather than grouping features into shared tiers: 1=suggestions, 2=comments, 3=trains, 4=routes/conductors, 5=settings/email-errors, 6=users (main.go route-registration comments, admin_base.html nav gating). A new moderation queue should claim the next unused integer (used 7 for vantage_spots), not piggyback on an existing level.
- **Templates auto-register from the embedded FS by filename** — main.go's template-loading loop (`embedFS.ReadDir("templates")` / `"templates/admin"`, ~main.go:435-469) picks up any `.html` file dropped into `templates/` or `templates/admin/` automatically; the map key used by `renderPublic`/`renderAdmin` is the literal filename. No explicit registration list to maintain.
- **Every inline `<script>` needs `nonce="{{.Nonce}}"`** — `script-src` has no `'unsafe-inline'`, so a forgotten nonce silently no-ops the script rather than erroring loudly.
- **New public-submission flows should reuse the existing anti-abuse primitives wholesale**: `setTimingCookie`/`checkTiming` (≥2s), the honeypot triad (`website` field + obfuscated `a`/`b` fields set by inline JS), `hashIP`, and `checkSuggestRateLimit`/`recordRateLimit` (validate.go) are all generic (keyed by `action='suggest'` in `rate_limit_log`), not specific to the original suggestions table.
- **`go build ./...`** is the correct full-repo build check (confirmed working command for this repo).

## GOTCHAS

- **Public-page CSP `img-src` does NOT include `cdn.jsdelivr.net`**, even though `script-src`/`style-src` do (main.go `securityHeaders`, public branch ~main.go:577-596). Leaflet's default `L.marker()` icon is an image hosted alongside `leaflet.min.css` on jsdelivr — so any plain `L.marker()` on a public page renders as a broken image, silently, with no console CSP violation obvious to a casual look. The existing code already worked around this on `/map` by using `L.circleMarker` (no image) for live-train dots; new marker UI should follow that pattern or use `L.divIcon` with inline HTML (no external asset) rather than the default icon.
- **No `connect-src` in the CSP** means it falls back to `default-src 'self'` — same-origin `fetch()` calls need zero CSP changes, but any future third-party API call from client JS would need an explicit `connect-src` addition.
- **Struct methods with pointer receivers can't be called directly from a `{{range}}` value in html/template** (the range variable isn't addressable via reflection). `Media.PublicURL()`/`HasGeo()` and `Suggestion.StatusLabel()` are pointer-receiver methods that are only ever invoked through funcMap wrapper functions (`main.go` funcMap: `mediaURL`, `hasGeo`, etc.), never as `{{.PublicURL}}` directly in a template. A new struct that needs `{{.Method}}` called inline in a template range must use a **value receiver**, not pointer.
- **`processUpload`'s (media.go) filename-uniqueness check is hardcoded to the `media` table** (`SELECT COUNT(*) FROM media WHERE stored_filename LIKE ?` inside `nextFilename`). Reusing `processUpload` for a different table's uploads means that count is meaningless for collision avoidance — safe reuse requires passing a naturally-unique `entitySlug` per call (e.g. incorporating a fresh random token) rather than trusting the count logic.
- **`maybeNotifyPending()` / `pendingItemsTotal()` (notify.go) hardcode the exact list of tables considered "pending review items"** (suggestions, comments, users, conductor_requests). Adding a new moderation queue does not automatically get counted — requires manually adding a `COUNT(*)` line to `pendingItemsTotal`.
- **Despite the standing project preference for `STRICT WITHOUT ROWID` on all new SQLite tables, none of the existing tables in this codebase use it** (checked full `applySchema` in db.go) — the convention hasn't been retroactively applied and appears to only bind future new tables, per the memory's own migration-safety caveat.
- **`WITHOUT ROWID` tables can't have an autoincrementing `INTEGER PRIMARY KEY`** (no rowid alias to autoincrement). The codebase's existing workaround for this exact constraint is the `sessions`/`user_sessions` pattern: `id TEXT PRIMARY KEY` populated in Go via `newToken()` (64 lowercase hex chars from `crypto/rand`, main.go ~540) — the right model to copy for any new `STRICT WITHOUT ROWID` table that needs a generated identifier.
- **The admin UI version string is a hand-edited literal in `templates/admin_base.html:13`** (inside the nav `<span class="brand">`), not a Go constant — CLAUDE.md's instruction to "update the admin version" means editing that HTML line directly; there is no `AppVersion` symbol anywhere in the Go source.

## ARCHITECTURE

- Public vs. admin pages get **different CSP policies** from the same `securityHeaders` middleware, branched on whether the request path is under `app.adminCookiePath` (main.go ~577-596) — admin CSP is more permissive on `img-src` (`https: data: blob:`) than the public CSP's curated per-domain allowlist.
- `classifyPublicURL` (validate.go) is the single choke point for public URL submissions: it validates scheme/host against `allowedDomains`, normalizes YouTube/Vimeo URLs (stripping tracking params, converting `youtu.be`/Shorts to canonical `watch?v=` form), and returns `(domain, mediaType, normalizedURL, ok)` — four return values, easy to miscount when writing a new caller.
- `/map`'s Leaflet setup layers: `routeLayer` and `trainLayer` are `.addTo(map)` at creation (so their layer-control checkboxes start checked); any layer that should start **off** must be constructed as a bare `L.layerGroup()` without `.addTo(map)` and only registered in the `L.control.layers(...)` overlays object — Leaflet infers the checkbox state from whether the group is already on the map, not from any explicit "default" option.
